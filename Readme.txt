参数设置：
关于如何确定隐藏层的个数：Cybenko证明，三层网（一层隐藏层）如果有足够多的隐藏节点，可以逼近任何函数，所以，较少的隐藏层是可以胜任的。
关于如何确定每层神经元个数：理论上认为，节点数越多，效果越好，但是节点太多会导致训练很慢。具体的设置，看自己的实验效果吧。
学习步长和质量因子的选择，学习规则和激活函数的选择，如果没有特殊要求，按照默认就好。
主要事项：
在将数据代入网络时，一定要对数据进行归一化操作，最主要的是因为输出层的激活函数的阈值是有限制的，所以需要将目标数据映射到激活函数的值阈。例如这里使用双S型激活函数，所以需要映射到[-1,1]范围。
如果你采用的是线性激活函数，要将数据归一化到[0,1]区间，归一化方法  y=(x-min)/(max-min)
如果你采用的是非线性激活函数，要将数据归一化到[-1,1]区间，归一化方法  y=2(x-min)/(max-min)  -1
